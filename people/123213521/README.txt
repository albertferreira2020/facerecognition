# Instruções para uso da pasta

Esta pasta deve conter as imagens de referência da pessoa com ID: 123213521

## Como usar:
1. Coloque 2-5 fotos da pessoa nesta pasta
2. Use formatos: JPG, PNG, JPEG ou GIF
3. Certifique-se de que as fotos mostram claramente o rosto da pessoa
4. Evite fotos muito escuras, borradas ou com múltiplas pessoas

## Exemplo de arquivos:
- foto1.jpg
- foto2.png  
- selfie.jpeg

As fotos serão usadas como referência para comparar com novas imagens enviadas via API.
